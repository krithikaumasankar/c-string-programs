# C String Programs

A comprehensive collection of C programs demonstrating string manipulation, string algorithms, character processing, and standard library functions. The repository includes both manual implementations and programs using the C Standard Library (`string.h` and `ctype.h`).

---

## Repository Structure

```
c-string-programs
│
├── Basic_String_Operations
├── String_Analysis
├── String_Library_Functions
├── String_Algorithms
└── README.md
```

---

## Programs Included

### Basic String Operations

- Lowercase to Uppercase using `toupper()`
- Lowercase to Uppercase using ASCII values

---

### String Analysis

- Count Alphabets, Digits, Spaces and Special Characters using `<ctype.h>`
- Count Alphabets, Digits, Spaces and Special Characters using ASCII values
- Count Vowels, Consonants and Words
- Character Frequency (Occurrence) in a String

---

### String Library Functions

Demonstration of commonly used string functions:

- `strlen()`
- `strcpy()`
- `strcat()`
- `strcmp()`

> Additional functions such as `strrev()`, `strlwr()`, and `strupr()` are compiler-specific and may not be available in all C compilers.

---

### String Algorithms

- Palindrome String Check
- Alphabetical Sorting of Multiple Strings
